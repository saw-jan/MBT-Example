### Tools and Libraries:

- [Xstate](https://xstate.js.org/docs/) - _MBT library_
- [Playwright](https://playwright.dev/) - E2E testing library

### Install dependencies

```bash
npm install
# or
yarn install
```

### Start the App

```bash
npm start
# or
yarn start
```

### Run the test

```bash
npm test
# or
yarn test
```
